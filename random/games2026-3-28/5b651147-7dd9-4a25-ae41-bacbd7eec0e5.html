Description

PixiEditor is a free and open-source universal 2D editor. Create clean pixel-perfect sprites, paint with raster tools, or design crisp vector graphics — all on one canvas. Animate frame-by-frame or use the powerful Node Graph to build fully non-destructive visuals and custom effects.

It features a clean, familiar interface inspired by popular graphic editors, making it incredibly easy to jump in and start creating.

Whether you're sketching, animating, or crafting a game-ready asset, PixiEditor gives you the tools to bring your ideas to life.

Toolsets for any scenario

PixiEditor comes with three dedicated toolsets:

  • Pixel Art – Tools tailored for clean, pixel-perfect editing

  • Painting – Soft brush, basic painting tools, and anti-aliased shapes

  • Vector – Paths and shapes for scalable, resolution-independent artwork

All toolsets can be used on one canvas. Export to PNG, JPG, SVG, GIF, MP4, and more!

Animations

Create smooth frame-by-frame animations, or use nodes to animate custom shaders. Vector-based keyframe animation is also on our roadmap.

Node Graph

The Node Graph powers PixiEditor’s most advanced features. Every effect, layer, and structure is built using nodes or node connections.

You can customize every document non-destructively, combining pre-built effects and custom shaders to create stunning visuals and procedural art.
